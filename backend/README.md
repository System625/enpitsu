# Backend

This directory contains the backend service for enpitsu.

## Expected Structure

```
backend/
├── requirements.txt     # Python dependencies (or package.json for Node)
├── main.py              # App entrypoint (or index.js)
└── routes/              # API route handlers
```

## Setup

**Python:**
```bash
pip install -r requirements.txt
python main.py
```

**Node:**
```bash
npm install
node index.js
```

Runs on `http://localhost:8000` by default.
