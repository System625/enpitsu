
So here is what we are trying to build
A live agent that creates comic panels for you, manga, manhwa, american comic etc
Using the gemini live api of course, but there are a lot of bugs
We want it to be able to talk with us, like your ai companions, but this will be making comics in real time as you are speaking to it
You should be able to interrupt it and make it correct a particular page it already did
It should be able to play music while it's working, because you know, image takes time to create
Like what gemini said:

* Interactive Storyboarding: The user uploads the PDF, and the Agent immediately starts talking to them (using audio/voice). The Agent says, "I've read your story. For the first fight scene, should we go with a gritty Manhwa style or a classic 90s Manga layout?"

* Real-time adjustments: The agent generates the first few panels on the screen. The user can use their microphone to interrupt and say, "No, change the villain's hair to silver and make the panel read right-to-left." The agent immediately updates the visual on the screen.

* Multimodal Experience: Combine text (the story), images (the comic panels), and audio (the agent discussing the creative process with you) to create a deeply immersive app.
